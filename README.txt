CSSC HTTP Proxy Server
Protocolos de Comunicación, Segundo Cuatrimestre, 2010.

Autores:
Colotto, Matías Ezequiel
Cura, María Eugenia
Samra, Santiago José
Scaruli, Jorge Ezequiel


Esta es una implementación de un Proxy HTTP, siguiendo los lineamientos y especificaciones de la RFC 2616. 
Tiene algunos agregados, tal como especifica el enunciado provisto por la cátedra.
